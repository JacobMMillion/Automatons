Welcome to the Dashboard

This dashboard provides quick access to various types of data analysis. Below is a summary of each section:

Video-Data

This tab allows an associate to enter a URL to view post statistics on a day-by-day basis, visualized with a graph. Use it to quickly assess video performance.

Trial-Data

Similar to Video-Data, this section provides day-by-day visualizations for trials per app. It helps you monitor and compare trial activities over time.

Video-Search

Use Video-Search to find specific video data based on parameters such as post date, URL, creator, and more. It provides a targeted approach for data retrieval.
